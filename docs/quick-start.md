# Швидкий старт

За кілька хвилин ви підключите свої дані, один раз опишете контекст — і зможете
користуватися готовими промптами з цього сайту.

## 1. Підключіть джерела

Конектори відкривають асистенту доступ до ваших робочих даних — він читає їх
напряму, без ручного копіювання.

| Конектор | Що він відкриває |
| --- | --- |
| Gmail | Пошта: листи, треди, вкладення |
| Google Calendar | Зустрічі, події, розклад |
| Google Drive | Файли й папки |
| Google Docs / Sheets | Документи й таблиці |
| HubSpot | CRM: клієнти, угоди, контакти |
| Excel | Локальні таблиці |

> **Порада:** не треба підключати все одразу — почніть з 1–2 джерел, якими
> користуєтесь щодня (наприклад, Gmail і Calendar), а решту додасте потім.

## 2. Дайте інструкцію

Асистент працює краще, коли ви один раз описали контекст — далі він враховує
його в кожній відповіді.

Опишіть коротко:

- **Хто ви** — роль, зона відповідальності.
- **Компанія** — чим займається, продукт, ринок.
- **Пріоритети** — що зараз найважливіше.
- **Тон листів** — як ви спілкуєтесь із клієнтами й командою.

Це разова дія: один абзац контексту економить десятки уточнень пізніше.

## 3. Як користуватися промптами з цього сайту

1. **Розгорніть картку** промпту, який вам потрібен.
2. **Натисніть копію** — текст промпту одразу в буфері обміну.
3. **Вставте у ChatGPT або Claude** з підключеними конекторами й запустіть.

> Щоб задача виконувалася щодня автоматично — див. розділ **Регулярні задачі**.
